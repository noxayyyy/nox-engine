\doxysubsection{Detailed Description}
Manages sprite sheet animations and transitions between them using a state machine. 

This component controls an entity\textquotesingle{}s sprite to play different animation sequences. It uses an adjacency matrix to define transitions (Edges) between animations based on conditions.
